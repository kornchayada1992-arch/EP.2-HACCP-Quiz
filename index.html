🔗 ตั้งค่า Google Sheets

ทำครั้งเดียว ผลทุกคนจะไหลเข้า Spreadsheet อัตโนมัติ ไม่ต้อง Google Form แยกต่างหาก

📋 ขั้นตอน (ใช้เวลา ~8 นาที):

1. เปิด Google Sheets → สร้างไฟล์ใหม่
2. ไปที่ Extensions → Apps Script
3. ลบโค้ดเดิมทั้งหมด แล้ววางโค้ดนี้:
function doPost(e) { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sh = ss.getSheetByName("ผลแบบทดสอบ") || ss.insertSheet("ผลแบบทดสอบ"); if (sh.getLastRow() === 0) { sh.appendRow(["วันที่","เวลา","ชื่อ-นามสกุล","แผนก","รหัสพนักงาน", "คะแนน","เปอร์เซ็นต์","ผลการทดสอบ", "ข้อ1","ข้อ2","ข้อ3","ข้อ4","ข้อ5"]); sh.getRange(1,1,1,13).setFontWeight("bold") .setBackground("#D62828").setFontColor("white"); } var d = JSON.parse(e.postData.contents); sh.appendRow([d.date,d.time,d.name,d.dept,d.empCode, d.score,d.percent,d.result, d.ans1,d.ans2,d.ans3,d.ans4,d.ans5]); return ContentService.createTextOutput("OK") .setMimeType(ContentService.MimeType.TEXT); } function doGet(e) { return ContentService.createTextOutput("Connected!") .setMimeType(ContentService.MimeType.TEXT); } 4. กด Deploy → New deployment
5. Type: Web app | Execute as: Me | Access: Anyone
6. กด Deploy → คัดลอก Web app URL มาวางด้านล่าง

กรณีใบมีด / คัตเตอร์แตกหักระหว่างใช้งาน
⚠️ ถือเป็น Food Safety Incident
เมื่อพบใบมีดหรือคัตเตอร์แตก หัก หรือชำรุด ระหว่างกระบวนการผลิต ถือว่าเป็นเหตุการณ์ที่ส่งผลต่อความปลอดภัยด้านอาหาร (Food Safety Incident) และต้องดำเนินการทันทีตามระบบ HACCP โดยไม่รอช้า
⛔ Food Safety Incident ⛔ Physical Hazard ✅ ต้องปฏิบัติตาม HACCP ทันที
พนักงานต้องปฏิบัติ 6 ขั้นตอน
1
🚫

หยุดการผลิตทันที

หยุดเครื่องจักรและหยุดผลิตทันที ห้ามปล่อยให้กระบวนการดำเนินต่อไปโดยเด็ดขาด

2
📣

แจ้งหัวหน้างาน

แจ้งหัวหน้างาน / หัวหน้าฝ่ายผลิตทันที เพื่อตัดสินใจและดำเนินการต่อไป

3
📦

กักกันผลิตภัณฑ์

กักกันผลิตภัณฑ์และวัตถุดิบทั้งหมดในล็อตที่เกี่ยวข้อง รอการตรวจสอบ

4
🔍

ค้นหาและเก็บชิ้นส่วน

ค้นหาและเก็บเศษใบมีดให้ครบถ้วน 100% ส่ง QC ตรวจนับ บันทึกจำนวนชิ้นส่วนทุกชิ้น

5
🧹

ทำความสะอาดพื้นที่

ทำความสะอาดพื้นที่และอุปกรณ์อย่างละเอียด ก่อนเริ่มงานใหม่ทุกครั้ง

6
📝

บันทึกและป้องกันซ้ำ

บันทึกเหตุการณ์ วิเคราะห์สาเหตุ และดำเนินการป้องกันการเกิดซ้ำลงในเอกสาร

❗ ห้ามเริ่มการผลิตต่อ จนกว่าความเสี่ยงจะถูกควบคุมและได้รับการอนุญาตจากหัวหน้า
ข้อห้ามในพื้นที่การผลิต (Preventive Control)
🚫

ห้ามนำหมุดเข็ม / หมุดโลหะ / วัตถุแหลมคมทุกชนิด
เข้ามาในพื้นที่การผลิต

ข้อห้ามนี้บังคับใช้กับทุกคนที่เข้าพื้นที่ผลิต โดยไม่มีข้อยกเว้น
⚠️
เป็น Physical Hazard ที่มีความเสี่ยงสูง วัตถุแหลมคมขนาดเล็กสามารถปนเปื้อนในผลิตภัณฑ์ได้โดยไม่รู้ตัว ส่งผลให้ผลิตภัณฑ์ไม่ปลอดภัย
🔍
ไม่สามารถควบคุมการสูญหายได้ หมุดเข็มและวัตถุโลหะขนาดเล็กยากต่อการตรวจพบและติดตาม หากหลุดหายในพื้นที่ผลิต
📋
ไม่สอดคล้องกับมาตรฐาน GHP / HACCP / FSSC 22000 มาตรฐานความปลอดภัยด้านอาหารระดับสากลห้ามนำวัตถุอันตรายเข้าพื้นที่ผลิตโดยเด็ดขาด
✅

ใช้อุปกรณ์ที่ปลอดภัยเท่านั้น

อนุญาตให้ใช้ แม่เหล็ก / คลิปหนีบกระดาษแบบโลหะ ที่สามารถตรวจจับและติดตามได้เท่านั้น
หากไม่แน่ใจ ให้ถามหัวหน้างานก่อนนำเข้าพื้นที่ทุกครั้ง

🎯 Food Safety คือความรับผิดชอบของทุกคน
พบความเสี่ยง = ต้องหยุด และแจ้งทันที

🛡️ HACCP คืออะไร?
Hazard Analysis and Critical Control Points — ระบบการจัดการความปลอดภัยด้านอาหาร ที่วิเคราะห์อันตรายที่อาจปนเปื้อนในผลิตภัณฑ์ และกำหนดจุดวิกฤต (CCP) ที่ต้องควบคุมเพื่อป้องกันก่อนเกิดขึ้น ในโรงงานของเรามี 2 จุด CCP หลัก
CCP1-P · Slitting Process CCP2-P · Re-Slit Process
